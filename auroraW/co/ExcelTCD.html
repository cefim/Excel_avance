Consignes _ Récupération de données

Télécharger, ranger et utiliser le fichier ressource sur le lien suivant :

TCD_colocataires.xlsx

Consignes _ Créer un tableau croisé dynamique

Positionner le rectangle de sélection dans la plage de cellule contenant le détail du budget courses des colocataires

Puis cliquer dans l'onglet Insertion=>Tableau croisé dynamiques

Vérifier que la sélection se porte bien sur toute la plage de cellules contenant les données des colocataires (attention aux lignes vides qui peuvent stopper la sélection de toute une plage)

Choisir ensuite ou le tableau croisé dynamique va être insérer (Nouvelle feuille ou Feuille existante)

Choisir quelles données doivent apparaître dans le tableau en cochant et déplaçant les étiquette : "nom", "montant", "date" et "courses"

Positionner maintenant "nom" et "date" dans les lignes, "courses" dans les filtres, "montant" doit se mettre par défaut dans Valeurs

Sélectionner maintenant dans la liste déroulante "courses", apparaissant en haut du tableau, l'un des différents éléments : "Boissons", "Légumes", "Surgelés" ou "Viande" afin de filtrer les résultats affiché dans le tableau.

Consignes _ Créer un graphique croisé dynamique

En suivant la même méthode que pour le tableau croisé dynamique, insérer un graphique croisé dynamique que l'on peut trouver dans l'onglet Insérer=>Graphique croisé dynamique
